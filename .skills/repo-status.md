# repo-status Skill

Auto-updates README and TIL index.

## Triggers

Run when files added to: `leetcode/`, `HackerRank/`, `CodeWarsSolutions/`, `learning-notes/TIL/`, `workbooks/`, `miniprojects/`

## Actions

1. **README counts**: Update table with file counts
2. **TIL index**: Group TILs by category (Git, CLI/System, Media, Development)
3. **Recent activity**: Show last modified files

## Usage

Load skill when user asks:
- "Update README"
- "Update TIL index"
- "Show current state"

## Categories

| Category | Keywords |
|----------|----------|
| Git | git clone, git fetch |
| CLI/System | xrandr, xinput, nmcli, du, bash, terminal |
| Media | ffmpeg, yt-dlp, video |
| Development | ankiconnect, ripgrep, python, api |
