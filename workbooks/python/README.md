# Python Workbooks

Small, self-contained Python problem sets for building fluency in core language features —
the stuff that gets skipped when you live in frameworks.

Each file:
- Has a short concept summary at the top
- Contains 3–5 problems with `TODO` stubs
- Is runnable: `python3 generators.py` runs assertions and prints pass/fail

## Structure

```
foundation/     # Core Python idioms (pre-intermediate)
pythontips/      # One file per chapter of https://book.pythontips.com
```

## Running

```bash
# Single workbook
python3 foundations/comprehensions.py

```

## Progress

Mark a file done by replacing `TODO` with your solution. Commit as you go.
