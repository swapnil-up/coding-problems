My mind is blown today. I used the Gemini CLI, and I sort of get it all now. I used it to go fix my repository, which had some rule and permission problems. It made a lot of file changes, though I still had to manually put in permission checks for each of the buttons. It handled a good portion of that before it ran out of credits.

Then, on a completely free model from OpenCode, I was able to set up a really solid developer script that handles both setup and running the project for me. For my specific use case, I added a lot of quality-of-life features switched to uv, added linters, formatters, pre-commit hooks. It was great.

I’m genuinely excited to let it loose on my dev setup repository. There are so many optimizations there that I want to explore and implement.

At the same time, there’s almost a sense of nihilism too a kind of localized existential dread. It feels like my field is in for massive changes. Programming as it is may not really exist in a few years. It’s like I’m learning assembly at a time when compilers are just starting to become mainstream.

Nepal’s probably safe for a bit longer, but I’m talking about programming as a craft. It feels very likely that coding, as I know it the version where we grind through tutorials and build five different projects just to understand the common patterns won’t exist in the same way.

These days, it feels like the most popular “programming language” is English, and honestly, it kind of is.

I know one foot of mine is going to move toward learning AI inference, compute costs, building LLMs, and so on. But the other foot, the one that actually loves building things, getting into the nitty-gritty the compilers, interpreters, programming languages, classical problems that part now feels like it might soon become more of an amusing pastime.

Like Sudoku. Something niche. Something people do for the love of it.
